console.frog
============

A bad pun for your JavaScript console, inspired by a commit message from @ConradIrwin.

```
.          (ribit)
             /
      o  o
     ( -- )
  /\( ,   ,)/\
^^   ^^  ^^   ^^
```


Installation
------------

### Bower

```shell
bower install --save console-frog
```

### Node

```shell
npm install --save console-frog
```


Usage
-----

```javascript
require("console-frog");
console.frog();
```
